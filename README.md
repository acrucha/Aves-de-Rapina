## Como jogar?

1. Clone este repositório:
```
  git clone 'https://github.com/acrucha/Aves-de-Rapina'

```
2. Faça as instalações necessárias:

```
    sudo apt install love

```

3. Entre na pasta "Aves-de-Rapina"

```
    cd Aves-de-Rapina

```

4. Compile e rode o jogo:

```
    love .

```
